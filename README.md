# Leone Meccanica

Project Template by AWS CloudFormation

## Overview

### aws/env

Environment variables for CloudFormation

### aws/scripts

Create or Update CloudFormation stack

### aws/templates

CloudFormation templates

## Requirements

### System / Tools

- direnv

## Setup

### Activate direnv

```sh
$ cp .envrc.origin .envrc
$ direnv allow
```
